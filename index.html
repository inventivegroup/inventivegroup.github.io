Welcome to the Inventive R.P.S. Coding Contest

aka Rock Paper Scissors

Instructions

  1. Pair up (or trios)! Pick an IDE and code share - like LiveShare with VS Code. Name your team/project. This will be the name of your folder.
  2. One of you needs to clone the github repo. Create a folder for your project that is your team name.
  3. Create the best Rock Paper Scissors game you can. You can dress it up however you want, but it must have the following features/rules:
    • Each team can pick the language/library to use.
    • When the user clicks an option, a computer option should be shown or you can play another user
    • There are 5 rounds to the game, each round should show who won the round
    • After all 5 rounds are played a message should showcase the winner.
    • After the game has been played a restart button should appear to restart the game
    • You have 45 minutes to complete this exercise
    • It should work and look good!
    • Voting will be based on code cleanliness, code efficiency, and creativity on the asthetics.
  4. When done, push your code to the main branch when you are ready and it will publish to this site.
  5. Be sure to add your names below and the link to your project homepage
  6. We will then do a brief show and tell and send the link to the general channel for voting.
